# Boids Algorithm Research

The Boids algorithm was developed by Craig Reynolds in 1986.

It models flocking behaviour using three rules:

## Separation

Avoid collisions with nearby drones.

## Alignment

Match velocity with nearby drones.

## Cohesion

Move toward the average position of neighbours.

## Application

These rules could allow security drones to:

- Maintain coverage
- Avoid collisions
- Operate without central control
