# Drone Swarm Security System

## Overview

The Drone Swarm Security System is an autonomous surveillance concept that uses multiple UAVs working cooperatively through swarm intelligence principles.

Unlike traditional fixed CCTV systems, the drones can dynamically reposition themselves, track areas of interest, and maintain surveillance coverage using coordinated autonomous behaviour.

This project combines concepts from:

- Aerospace Engineering
- Artificial Intelligence
- Swarm Intelligence
- Autonomous Systems
- UAV Technology
- Computer Vision

---

## Problem

Traditional security cameras suffer from:

- Fixed viewing angles
- Limited coverage areas
- Single points of failure
- Poor adaptability to changing environments

This project aims to investigate whether a coordinated drone swarm can provide a more flexible and resilient security solution.

---

## Proposed Solution

A fleet of autonomous drones operating as a distributed surveillance network.

The drones would:

- Patrol predefined areas
- Maintain formation using flocking algorithms
- Share information between units
- Automatically reposition when threats are detected
- Cover blind spots dynamically
- Continue operating if individual drones fail

---

## Key Technologies

### Swarm Intelligence

The drones cooperate using decentralised decision making.

### Flocking Algorithms

Based on three primary behaviours:

1. Separation
2. Alignment
3. Cohesion

### Artificial Intelligence

Potential future applications:

- Object detection
- Intrusion detection
- Threat classification

### UAV Systems

- Flight control
- Navigation
- Communication systems
